# Handoff Prompt

Update the project handoff.

Read:

- docs/current-state.md
- docs/tasks.md
- docs/decisions.md
- docs/handoff.md

Then update:

- docs/current-state.md
- docs/tasks.md
- docs/decisions.md
- docs/handoff.md

The handoff must include:

1. Last completed work
2. Current state
3. Current sprint
4. Next recommended action
5. Known blockers
6. Known risks
7. Files recently modified
8. Any important implementation notes
9. Any decisions made
10. Suggested next prompt to run

Do not code.
